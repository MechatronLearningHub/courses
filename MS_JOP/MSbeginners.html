AZ 104 - Azure Administrator (Learner To Administrator) Course at Mechatron Learning Hub

If you’re an administrator responsible for deploying and managing Microsoft 365 in cloud and hybrid environments, Mechatron Learning Hub’s Microsoft 365 Administrator course is perfect for you.

As a Microsoft 365 administrator, you:

  • Function as the integrating hub for all Microsoft 365 workloads.
  • Coordinate across multiple Microsoft 365 workloads.
  • Work with architects and other administrators responsible for workloads, infrastructure, identity, security, compliance, endpoints, and applications.
    • This exam measures your ability to accomplish the following technical tasks: deploy and manage a Microsoft 365 tenant; implement and manage Microsoft Entra identity and access; manage security and threats by using Microsoft 365 Defender; and manage compliance by using Microsoft Purview.

What are you going to learn?

Course Prerequisites

  • Basic understanding of computing concepts, including what a virtual machine is and what an operating system is?
  • Familiarity with the concept of networking, such as what the Internet is and how a router works.
  • General knowledge of what software applications are and how they work on a basic level.
  • An interest in learning about cloud services and Microsoft Azure.
  • Microsoft Azure Fundamentals: Describe cloud concepts
  • Azure Fundamentals: Describe Azure architecture and services
  • Azure Fundamentals: Describe Azure management and governance
  • Ability to understand and engage with the training in English.

Major topics covered

AZ - 104 Azure Administrator (Learner to Administrator)

  • Learn to manage identities and access using Azure Active Directory.
  • Azure Architecture Overview.
  • Understand Azure governance, compliance policies, and resource management.
  • Set up and manage virtual networking and intersite connectivity for secure infrastructure Optimize network traffic and implement efficient Azure storage solutions.
  • Explore serverless computing with Azure services and enhance your skills in data protection and monitoring.
  • Our Azure 104 online training prepares you to apply these concepts in real-world scenarios, equipping you for challenges in the Azure ecosystem.
  • Azure Management Tools & Licenses.
